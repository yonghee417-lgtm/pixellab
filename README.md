# PixelLab (픽셀랩)

> 사진 편집 · 그래픽 디자인 도구 — Photoshop-lite. 윈도우/맥 동시 지원.

검정 + 골든 망고 액센트의 다크 UI. 같은 브랜드 라인의 영상편집기 [FrameLab](https://github.com/) 과 룩을 통일했습니다.

## 기능

- **캔버스**: 인스타/스토리/유튜브/A4/배경화면/4K 등 프리셋 + 커스텀 사이즈, 투명 배경 지원
- **레이어 시스템**: 추가 / 삭제 / 순서 / 잠금 / 숨김 / 불투명도 / 블렌드 모드 16종
- **도구**: 이동·선택·자르기·텍스트·사각형·원·선·다각형·브러시·지우개·매직지우개·스포이드·마스크
- **이미지 편집**: 가져오기 / 리사이징 / 회전 / 좌우·상하반전 / 자르기
- **배경 제거**: `@imgly/background-removal` 로 완전 로컬 처리 (영구 무료)
- **필터**: 밝기·대비·채도·색조·블러·비네팅 + 흑백·세피아·반전 + 9종 프리셋
- **텍스트**: 30+ 상업 사용 가능 한글/영문 폰트, 그림자·외곽선·자간·줄간격
- **히스토리**: 최대 50단계 undo/redo (Ctrl+Z / Ctrl+Y)
- **저장**: 자동 저장 + 다른 이름으로 저장 (`.pxlab`) + PNG/JPG/WebP 내보내기 (1x/2x/3x)

## 설치 / 실행

```bash
npm install
npm run fonts        # 상업 사용 가능 무료 폰트 다운로드 (assets/fonts)
npm run dev          # 개발 모드
npm run build        # 패키지 빌드 (release/)
```

## 단축키

| 카테고리 | 단축키 |
|---|---|
| 도구 | V 이동 · M 선택 · C 자르기 · T 텍스트 · R 사각형 · O 원 · L 선 · B 브러시 · E 지우개 · I 스포이드 · H 손바닥 |

## 프로젝트 파일 (.pxlab)

JSON 텍스트 파일로 Fabric.js 직렬화 결과 + 레이어 메타 + 캔버스 설정을 담습니다.
이미지 데이터는 dataURL로 임베드되어 단일 파일에서 모든 자료를 갖고 다닙니다.

## 라이선스

이 앱에 포함되는 모든 폰트는 SIL OFL 또는 자체 무료 라이선스로 상업 사용이 가능합니다.
